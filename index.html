Conferência Python Espírito Santo 2017

Brasil - 29 de Abril de 2017

A Conferência

A Conferência Python Espírito Santo 2017 reúne um grupo de pessoas durante todo o dia para discutir sobre software livre, práticas de desenvolvimento e Python. Cada palestra terá 40 minutos, sendo 30 para a palestra e 10 para perguntas e elas serão aceitas até o dia 22 de Abril de 2017.

Garanta a sua vaga!

Agenda

08:00h - Reception
08:20h - Breakout 1

Palestrante: Pedro Moreno

Introdução ao Python

Mostrar fundamentos da programação e como a linguagem python viabiliza os elementos em uma linguagem dinâmica com tratamento de erro de excelente qualidade com o uso das exceptions e explicando o "duck type" e finalizando com comandos básicos para auto conhecimento no python como dir() e help().

09:10h - Breakout 2

Palestrante: Gabriel Cypriano Saca

Pacotes e Ferramentas Python para Data Science e Machine Learning

Introdução aos pacotes e ferramentas Python utilizados em Data Science e Machine Learning, como por exemplo Jupyter Notebook, NumPy, SciPy, Matplotlib, Pandas e Scikit-learn.

10:00h - Breakout 3

Palestrante: Fabricio Vargas Matos

Sistemas de recomendação de conteúdo usando Machine Learning e TensorFlow

O objetivo é mostrar para programadores sem conhecimento prévio em ML e TF como, em linhas gerais, usar TensorFlow para implementar um sistema de recomendação eficiente e escalável. O algoritmo implementado seria um 'Model Based Collaborative Filtering' que agrupa os items recomendados em 'Latent Factors', identificados dinamicamente via 'Gradient Descend Optimization'.

10:50h - Lightning Talks
11:40h - Lunch
14:00h - Breakout 4

Palestrante: Guilherme Mauro

Análise e Processamento de Imagens com OpenCV

Conceitos básicos do módulo OpenCV e suas funções. Serão apresentados exemplos práticos e desenvolverei uma aplicação durante a palestra.

14:50h - Breakout 5

Palestrante: Bernardo Fontes

TDD com Mock e Orientação a Objetos

Não podemos nos esquecer que teste também é código! Nessa apresentação falo sobre como podemos utilizar a estratégia do uso de Mocks para a escrita de testes mais assertivos e simples.

15:40h - Breakout 6

Palestrante: Rafael dos Santos

Azure + Python - Usando o poder da nuvem para expandir sua aplicação

Mostrar como os padrões de desenvolvimento em nuvem junto com os benefícios do Microsoft Azure podem dar um upgrade em aplicações Python, desde simples protótipos ate aplicações de escala global.

16:30h - Lightning Talks
17:20h - Closing
